# FTS Evals Bulk Actions Prototype

Interactive HTML prototypes for FTS Evals Bulk Actions features.

## Prototypes

1. **FTS-Evals-Bulk-Actions-Prototype.html** - Main prototype with all features:
   - User Management page with search, filter, bulk invite, and export
   - Attempt Management page with attempt statistics
   - Re-invite functionality for completed attempts
   - Status statistics cards

2. **FTS-Evals-Bulk-Actions-Prototype-PMUS06.html** - Prototype specifically for PMUS06 (Re-invite feature)

3. **FTS-Users-Page-Prototype.html** - Users page prototype with search and filter functionality

## Features

- **PMUS01**: Filter users by email on User Management page
- **PMUS02**: Search users on main Users page
- **PMUS03**: Export invite statuses to CSV
- **PMUS04**: Bulk invite sending with enhanced pagination
- **PMUS05**: Status statistics overview on both User Management and Attempt Management pages
- **PMUS06**: Re-invite for retake at user level

## Usage

Open any of the HTML files in a web browser to interact with the prototypes. All prototypes are self-contained with embedded CSS and JavaScript.
